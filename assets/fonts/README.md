# Fonts

This directory should contain the Lora font files.

## Required Files

Download Lora from Google Fonts: https://fonts.google.com/specimen/Lora

Download the following weights and place them in this directory:
- `Lora-Regular.ttf`
- `Lora-Medium.ttf`
- `Lora-SemiBold.ttf`

## Installation Instructions

1. Visit https://fonts.google.com/specimen/Lora
2. Click "Download family" to get all weights
3. Extract the downloaded ZIP file
4. Copy the following files to this directory:
   - `static/Lora-Regular.ttf` → `Lora-Regular.ttf`
   - `static/Lora-Medium.ttf` → `Lora-Medium.ttf`
   - `static/Lora-SemiBold.ttf` → `Lora-SemiBold.ttf`

The application will automatically load these fonts at startup. If the fonts are not found, it will fall back to Georgia, then the system default serif font.
